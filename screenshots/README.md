## 스크린샷 촬영 방법

1. `example` 앱을 실행합니다:
   ```bash
   cd example
   flutter run
   ```

2. 각 기능을 테스트하며 스크린샷을 촬영합니다.

3. 스크린샷을 `screenshots/` 폴더에 위의 파일명으로 저장합니다.

4. 권장 해상도: 최소 400x800px (또는 더 높은 해상도)
